    Write inline markdown inside any HTML file.

    Inline Markdown

    This is an example of inline markdown. Just write markdown wherever you want and wrap it with {{#markdown}} {{/markdown}} section tags, then run grunt assemble to compile it to HTML.

    Embedded Templates!

    You can even add handlebars templates inside the markdown, like this:

    <nav class="nav nav-pills nav-stacked">
{{#page.links}}
  <li class="{{modifier}}">[{{text}}]({{href}})</li>
{{/page.links}}
</nav>

    and to get the data for our links we can either reference an external .json file, or use YAML front matter, like this:

    ---
  links:
    -
      href: markdown-inline.html
      text: Inline Markdown
      modifier: active
    -
      href: markdown-imported.html
      text: Imported Markdown
    -
      href: external-content.html
      text: External Content
    -
      href: grid.html
      text: Grid
    -
      href: structured-content.html
      text: Structured Content
---
